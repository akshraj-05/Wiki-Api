
# Wiki Api

this is the wikipedia **RESTful** api that Handles 

* get
* post
* put
* patch
* delete
Requests i used **postman** api tool for creating this api



## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
